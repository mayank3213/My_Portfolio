## 🧠 About This Portfolio

This is my personal portfolio website built with **React.js**, designed to showcase:

- 👨‍💻 Who I am
- 💼 Projects I've worked on
- 🧰 Tech stack I use
- 📫 Ways to contact me

It’s minimal, clean, fast, and responsive — designed with a developer-first approach in mind.

## 🔥 Features

✅ Fully responsive design  
✅ Smooth animations & transitions  
✅ Modern, minimal UI  
✅ Project cards with links  
✅ Contact form (EmailJS / Formspree)
